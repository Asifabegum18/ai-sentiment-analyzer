# 💬 AI Sentiment Analyzer

A professional AI-powered Sentiment Analysis web application built using Natural Language Processing (NLP) and Streamlit.

---

## 🚀 Features

- Analyze text sentiment in real time
- Detects:
  - Positive 😊
  - Negative 😡
  - Neutral 😐
- Interactive and modern UI
- Displays polarity and subjectivity scores
- NLP-based sentiment prediction

---

## 🛠️ Tech Stack

- Python
- Streamlit
- TextBlob
- Natural Language Processing (NLP)
- Git & GitHub

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Asifabegum18/ai-sentiment-analyzer.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd ai-sentiment-analyzer
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Download TextBlob Data

```bash
python -m textblob.download_corpora
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```
